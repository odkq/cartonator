==========
Cartonator
==========

.. image:: http://i.imgur.com/cTu12FS.png


Print random cards for Bingo using PostScript

.. code-block::
 usage: python cartonator [-h] [-i INCLUDE] [-e EXCLUDE] <file.ps>

 positional arguments:
  file                  File to write the Postscript to. If ommited, write to
                        stdout

 optional arguments:
  -h, --help
      show this help message and exit
  -i INCLUDE, --include INCLUDE
      include always the specified number
  -e EXCLUDE, --exclude EXCLUDE
      include always the specified number
